# HoloBuild
In monocular vision systems, the tracking of multiple objects is dysfunctional due to occlusions and truncation of the objects. Although stereo vision allows the inclusion of depth information in order to reduce these problems, current methods can be based on computationally intensive dense matching or 3D reconstruction
In this paper, a real time 2.5D stereo multi-object tracking structure is proposed which integrates lightweight stereo geometry with the sound tracker management. Stereo matching is done with the help of Direct Linear Transform-based triangulation of bounding box coordinates without extracting features and ensuring the good correspondence under geometrical constraints. Dual-tracker architecture will ensure that each stereo view has separate trackers so that when targets are not visible in one view, they are visible in the other view. The results of experiments related to the use of a refrigerator monitoring dataset reveal that the proposed Stereo SORT algorithm has a MOTA of 0.932 and an IDF1 score of 0.823, which are better than monocular trackers like OC-SORT and Byte Track. The system achieves a median error of depth of 50.1 mm and is able to run at 70 FPS on commodity hardware, proving efficient and robust stereo tracking, without appearance features.
Keywords: stereo vision; multi-object tracking; stereo matching; depth estimation; stereo tracking; tracker re-identification; occlusion and truncation treatment.
